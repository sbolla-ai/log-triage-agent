# Log Triage Agent

A learning project: an LLM-powered agent that inspects log files.
Built as part of the Agentic AI learning journey.

## Setup
- Python 3.10+
- `python -m venv .venv && source .venv/bin/activate`
- `pip install anthropic python-dotenv`
- Copy your Anthropic API key to `.env`
